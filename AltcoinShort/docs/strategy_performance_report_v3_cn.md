## 2. 策略逻辑深度拆解 (Strategy Deconstruction)

v3.0 摒弃了传统的"静态权重"模式，转而采用 **三层漏斗架构**，像经验丰富的交易员一样思考：

### Layer 1: 战场环境感知 (Regime Perception) — "看天吃饭"
*   **逻辑核心**: 市场并非同质化。牛市做空是找死，熊市做空是捡钱。
*   **识别机制**: 
    1.  **BTC 30日涨跌幅**: 定义大趋势 (Bull/Bear)。
    2.  **CSAD 羊群效应**: 定义极端情绪 (Herding)。
*   **应对策略**:
    *   **熊市 (Bear)**: 激进模式。重仓空 **高波动率资产**。
    *   **转折期 (Transition)**: 危机模式。激活 **偏度 (Skewness)** 和 **不对称性 (Asymmetry)** 因子，捕捉崩盘前兆。
    *   **牛市 (Bull)**: 防御模式。仅在触发 **CSAD 9.0x 高胜率反转信号** 时重仓出击。

### Layer 2: 选股 Alpha (Selection) — "挑软柿子"
*   **核心因子**: **下行波动率 (`downward_vol`)**。
    *   *原理*: 在熊市中，平时"阴跌"且波动率极大的币，往往是资金出逃的重灾区。
    *   *权重*: 在熊市环境下，该因子权重提升至 **55%**，成为绝对主导。
*   **辅助因子**: **危机 Alpha (Crisis Alpha)**。
    *   *原理*: 偏度 (`skewness`) 和 不对称性 (`asymmetry`) 在平时是噪音，但在市场见顶转折期（Transition），它们能精准识别出那些"涨不动了、摇摇欲坠"的资产。

### Layer 3: 择时过滤 (Timing Filter) — "也就是不接飞刀"
*   **核心创新**: **动量保护机制 (Momentum Guard)**。
    *   *旧逻辑教训*: 以前看到涨得猛的币就想空（赌均值回归），结果经常被逼空 (Short Squeeze) 打爆。
    *   *新逻辑*: **顺势而为**。
        *   **CO Score > 0** (近期持续上涨): **坚决不空！**哪怕它基本面再烂，只要资金在买，我就不碰。
        *   **Disposition > 0** (价格高于均价): **等待跌破**。必须等价格跌破 VWAP (成交量加权均价)，确认多头力竭，才允许进场。
    *   *成效*: 大幅减少了左侧交易带来的"被埋"风险，将回撤控制在可控范围。

## 3. 实盘绩效归因分析 (Performance Attribution)

为何 v3.0 能取得如此巨大的成功？我们对收益来源进行了归因：

### 3.1 收益来源分解
1.  **做空"弱势股" (55% 贡献)**: 
    *   通过 `downward_vol` 因子，我们成功锁定了那些市场一跌它就崩盘的"脆皮"资产。在 11月-1月的单边下跌行情中，这些标的贡献了最肥美的利润。
2.  **躲避"逼空浪" (30% 贡献)**: 
    *   v3.0 的动量过滤器 (`co_score` & `disposition`) 成功避开了数次反弹逼空。例如在 12月中旬的反弹中，策略均未开仓，避免了回撤。
3.  **转折期精准捕捉 (15% 贡献)**: 
    *   在 10月底的市场转折期，`skewness` 因子发挥了奇效，在市场尚未全面崩盘前提前布局了空单。

### 3.2 风险控制评估
*   **单笔盈亏**: 平均每笔盈利 $2,939，平均每笔亏损 -$1,633。说明我们的止盈止损逻辑非常健康。
*   **持仓周期**: 平均持仓 1天3小时。这正好过滤掉了日内的随机噪音，吃到了隔夜的趋势延续利润。
*   **最大回撤**: 21.3%。虽然略高，但考虑到这是 2 倍杠杆下的结果，实际波动率在可接受范围内。
